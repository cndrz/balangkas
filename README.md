## Balangkas
An AI-powered thesis title generator and research companion. Enter a research topic or keywords, get three professional thesis titles with feasibility analysis, and surface relevant peer-reviewed literature — instantly.
Features

AI Title Generation — Generates 3 distinct, research-worthy thesis titles using Llama 3.3 70B via Groq
Feasibility Analysis — Each title includes a scope, complexity, and challenge assessment
Literature Finder — Pulls 3 relevant peer-reviewed studies from OpenAlex per selected title
Multi-page SPA — Home, About, and Help pages with client-side routing

## Tech Stack

Vite + Vanilla JavaScript
Groq API (Llama 3.3 70B)
OpenAlex API
Deployed on Vercel
